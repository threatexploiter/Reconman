# Reconman
## Use recon.sh 
### recon.py is under construction 

- You can use this code for a lot of things 
- To get burp config for manual enumeration and scope csv files of h1 programs
- Please paste your h1 program links in program_links.txt
- Update your discord webhook in WebAutomator.py if you want
- Run proglinkappender so that the burp and csv links are appended to links.json 
- After that run WebAutomator.py so that every program has its separate dir
- Before running recon.sh make sure to change ur TOOLSDIR variable to where all your tools are located

- Then run recon.sh To close out You can setup notify if you want
